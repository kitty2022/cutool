# cutool ✂️

一个简洁的在线图片切图工具，支持网格切图和自定义选区。

## 功能

- **网格切图** — 设置行数和列数，自动均分切片
- **自定义选区** — 鼠标拖拽框选区域，支持拖动/缩放/删除
- **输出尺寸设置** — 自定义切片输出宽高（0=原尺寸）
- **ZIP 打包下载** — 一键导出所有切片

## 使用

直接打开 `cutool.html` 即可使用，无需安装。

或通过局域网访问：
```
http://你的IP:3000/cutool.html
```

## 技术

纯前端 HTML + CSS + JavaScript，使用 Canvas API 处理图片，JSZip 打包导出。
